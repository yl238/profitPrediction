Files in this directory
------------------------------
First-Look.ipynb:            Jupyter-notebook to plot feature distributions and calculate stats

Feature-Extraction.ipynb:    Jupyter-notebook to test out functions for feature extraction and creation

Model-Tuning.ipynb:          Jupyter-notebook to tune the parameters of the ML models, calculate the classification reports and plots the ROC curve.

Maximise-Profit.ipynb: 	     Jupyter-notebook to build model to calculate the maximum profit.

-------------------------
estimating_uptake.py:	     Python script to run the classification and report of Logistic Regression, Random Forest and Gradient Boosting using the parameters tuned in the Model-Tuning.ipynb notebook. 

feature_engineering.py:      Python script containing methods for data conversion and feature engineering

analysis_utils.py: 	         Python script to plot lift curve and feature importance
